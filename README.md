## Cartesian Genetic Programming

**Configs**

The configs required are in the *configs.json* file. You can change them there.

**Requirements**

| Name       | Version |
|------------|---------|
 | python     | 3.10.4   |
| imageio    | 2.19.3  |
| numpy      | 1.23.1  |
 | matplotlib | 3.5.2   |

**Execute**

To execute the script from command line use:

`python main.py <path_to_configs_file>`
